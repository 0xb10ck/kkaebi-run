# 깨비런 (KKAEBI RUN)

한국 도깨비 설화와 오방색을 모티브로 한 탑다운 로그라이크 서바이벌 게임.

## 한 줄 소개
밀려오는 잡귀 무리를 도깨비의 신통(神通)으로 베어내며 살아남는, 한국풍 뱀파이어 서바이벌류 로그라이크.

## 장르
- 뱀파이어 서바이벌류 (Vampire Survivors-like) 로그라이크
- 탑다운 2D 액션 / 자동 공격 / 무한 웨이브 / 메타 프로그레션

## 엔진 / 기술 스택
- **엔진**: Godot 4.x
- **언어**: GDScript
- **렌더링**: 2D (Forward Plus / Compatibility, 웹 호환성 우선)

## 빌드 타겟
- **주 타겟**: HTML5 웹 (브라우저 플레이)
- 배포 형태: 정적 자산 호스팅 (itch.io / GitHub Pages 등)

## 컨셉: 한국 도깨비 × 오방색

### 세계관
플레이어는 산속 옛 사당에서 깨어난 어린 도깨비. 결계가 무너지면서 사방에서 몰려드는 잡귀(雜鬼), 객귀(客鬼), 처녀귀신, 그슨대 같은 한국 민속의 요괴들을 막아내야 한다. 방망이 한 자루로 시작해, 부적·삼지창·매구의 불·신장(神將)의 가호 등 한국 무속·설화의 신통력을 모아 강해진다.

### 오방색(五方色) 시스템
한국 전통 색채관에 기반한 5개 속성으로 무기와 적이 분류된다.

| 방위 | 색 | 오행 | 게임 내 속성 |
|------|----|----|-----|
| 중앙 | 황(黃) | 토(土) | 방어 / 결계 / 회복 |
| 동 | 청(靑) | 목(木) | 속도 / 관통 |
| 서 | 백(白) | 금(金) | 치명타 / 절삭 |
| 남 | 적(赤) | 화(火) | 화염 / 도트 데미지 |
| 북 | 흑(玄) | 수(水) | 둔화 / 광역 |

같은 색 무기를 모으면 진화하고, 상극 색을 조합하면 특수 합성 무기가 해금된다.

### 비주얼 톤
- 한지·먹·단청을 기반으로 한 2D 픽셀 아트 / 또는 평면 일러스트풍
- UI 모티브: 부적, 매듭, 단청 문양
- 사운드: 국악 타악(장구·꽹과리) + 신디 리드
