## 0. 사용 규약

### 0.1 명세 항목

각 보스(미니보스 포함)는 다음 7개 블록을 모두 포함한다.

1. **메타** — 한글명(영문 ID) / 출전 / 등장 챕터 / 등장 트리거
2. **외형 & 컨셉** — 픽셀아트 실루엣, 색감, 컷인 콘셉트
3. **기본 스탯** — HP / 방어력 / 이동속도 / 기본 공격력 / 사거리 / 충돌박스
4. **페이즈 구성** — HP 비율에 따른 페이즈 전환 조건, 페이즈별 키워드
5. **페이즈별 패턴** — 각 패턴마다 (a)발동 조건, (b)텔레그래프(예고 동작 + 지속시간), (c)데미지 판정 형태(원형/직선/광역/장판), (d)데미지량, (e)쿨다운
6. **페이즈 전환 연출** — 외형 변화 / 무적 프레임 / 컷인·VFX·카메라
7. **처치 보상 / BGM·SE 큐**

### 0.2 스탯 기준선

플레이어 표준값 (메타 강화 0레벨)

| 항목 | 값 |
|------|-----|
| 플레이어 HP | 100 |
| 플레이어 기본 공격력 | 10 (1초당 1회, 평균 DPS 10) |
| 플레이어 후반 평균 DPS(스킬 풀세팅 기준) | 60~120 |
| 플레이어 이동속도 | 120 px/s |
| 1프레임 | 1/60 s |

보스 HP 기준선은 "플레이어 평균 DPS 80 × 목표 처치 시간" 으로 잡는다.

| 단계 | 목표 처치 시간 | HP 기준선 |
|------|----------------|-----------|
| 1장 미니보스 | 25~40s | 1,800 |
| 1장 챕터보스 | 60s | 4,800 |
| 2장 미니보스 | 35s | 2,800 |
| 2장 챕터보스 | 90s | 8,000 |
| 3장 미니보스 | 40s | 3,800 |
| 3장 챕터보스 | 110s | 11,000 |
| 4장 미니보스 | 45s | 5,200 |
| 4장 챕터보스 | 130s | 15,000 |
| 5장 미니보스 | 50s | 7,000 |
| 5장 챕터보스 | 180s | 24,000 |
| 히든 챕터보스 | 150s | 변동(20,000±) |

방어력은 받는 데미지에 곱해지는 감산 비율로 정의: `최종 피해 = max(1, 입력 - 방어력×0.7)`.

### 0.3 단위

- HP, 데미지, 방어력, 경험치: 정수
- 이동속도: 픽셀/초 (px/s)
- 쿨다운, 텔레그래프, 무적 프레임: 초(s)
- 사거리/장판 반경/직선 길이: 픽셀(px)
- 충돌박스: 가로 × 세로 (px)
- 드랍률: 확률(%)

### 0.4 보스 공통 AI 상태

```
Spawn → CutsceneIntro → Idle → Pattern(N) → Recovery → Idle
                              ↘ PhaseTransition(무적, HP 잠금) → Idle(다음 페이즈)
                              ↘ Dying → DeathCutscene → Despawn
```

- **CutsceneIntro**: 등장 컷인(2.0~3.5s), 플레이어 무적/조작 불가.
- **Idle**: 0.5~1.5s 짧은 호흡. 페이즈별로 평균 Idle 길이가 짧아진다(긴장 상승).
- **Pattern 선택**: 사용 가능한(쿨다운 끝난) 패턴 중 가중치 랜덤. 직전 패턴은 가중치 -50%.
- **PhaseTransition**: 외형 변화 + 카메라 흔들림 + 보스 1.5~2.5s 완전 무적.
- **DeathCutscene**: 0.8s 슬로우 → 1.2s 폭발 → 보상 드랍.

### 0.5 텔레그래프 표기 규약

- **원형 장판**: 빨간 원 외곽선 → 안쪽이 점차 채워짐 → 채워지면 발동
- **직선 장판**: 빨간 띠 표시 → 사선 음영 채움 → 발동
- **광역(스크린)**: 화면 가장자리 빨간 비네팅 → 발동
- **유도탄/투사체**: 보스 위 0.3~0.6s "조준선" + SE 큐
- 텔레그래프 지속시간 = "예고가 시작되어 데미지 판정이 발생하기까지의 시간"

### 0.6 보상 공통 항목

모든 챕터보스 최초 처치 시 **신목의 이파리 ×1** 확정 드랍(§7.2). 미니보스는 신목 이파리 미지급. 도깨비 구슬은 처치 시 산정되어 런 종료 시 정산.

## 1. 미니보스 (5종)

각 미니보스는 스테이지 진행 3:30~4:00 시점에 등장(봉인석을 깨면 즉시 등장, 보상 1.5배). 페이즈는 2단(보통 70%/40%)으로 챕터보스 대비 1단계 단순화하지만, 텔레그래프/판정/쿨다운은 동일 규격으로 명시한다.

### MB01. 장산범 (Jangsanbeom / Jangsan Tiger-Spirit)

#### 1.1 메타
- **출전**: 부산 장산 일대 민간 전승 — 사람 목소리를 흉내 내 산속으로 끌어들이는 흰 털의 호랑이 요괴.
- **등장 챕터**: 1장 (두멍마을)
- **등장 트리거**: 1장 어떤 스테이지든 3:45 도달 또는 봉인석 파괴 시.

#### 1.2 외형 & 컨셉
- 키 1.8m 흰 털 호랑이, 붉은 눈, 입가 핏자국. 그림자가 사람 실루엣처럼 보이는 트릭 표현.
- 등장 시 "여보세요…" 사람 목소리 SE → 풀숲에서 흰 윤곽이 떠오름.

#### 1.3 기본 스탯
| 항목 | 값 |
|------|-----|
| HP | 1,800 |
| 방어력 | 2 |
| 이동속도 | 150 (도약 시 360) |
| 기본 공격력(접촉) | 14 |
| 충돌박스 | 64 × 40 px |
| 스프라이트 | 80 × 56 px |

#### 1.4 페이즈 구성
- **P1 (100%~40%)**: 유인기 — 가짜 아이템과 목소리로 함정 설치
- **P2 (40%~0%)**: 광폭화 — 도약 연타, 위장 해제

#### 1.5 페이즈별 패턴

**P1 패턴**

|------|-----------|-----------|-----------|--------|--------|
| 가짜 보물 미끼 | 입장 후 즉시, 이후 8s마다 | 화면 가장자리에 가짜 금화 아이콘 0.5s 표시 후 맵에 보물상자 스프라이트 출현 | 상자 반경 60px 원형 폭발(상자 접촉 시) | 18 | 8s |
| 사람 목소리 유인 | 플레이어와 거리 400px 이상 | 화면 밖에서 "여보세요" SE 1.0s + 미니맵 가짜 핑 | 없음(미끼) | - | 6s |
| 측면 급습 | 플레이어 화면 시야 밖에서 1.2s 대기 후 | 화면 가장자리에 흰 잔영 0.4s | 직선 돌진(길이 320px, 폭 60px) | 22 | 7s |
| 발톱 후리기 | 근접(120px 이내) | 앞발 들기 0.35s | 부채꼴 110도, 반경 100px | 16 | 2.5s |

**P2 패턴**

| 패턴 | 발동 조건 | 텔레그래프 | 판정 형태 | 데미지 | 쿨다운 |
|------|-----------|-----------|-----------|--------|--------|
| 광폭 도약 연타 | P2 진입 후 0.5s, 이후 6s마다 | 빨간 착지 원 3곳 순차 표시 0.4s씩 | 반경 80px 원형 × 3회 | 20 / 회 | 6s |
| 위장 해제 포효 | P2 진입 시 1회 + HP 20% 시 1회 | 보스 정지 + 흰 광채 0.6s | 광역 스턴 무효 + 보스 공속 +30%(영구) | 0 | 1회성 |

#### 1.6 페이즈 전환 연출
- 흰 털이 갈색으로 어두워지며 등에 핏줄 라인. 무적 1.2s.
- 화면 흔들림 약 0.3s, 호랑이 포효 SE.

#### 1.7 처치 보상 / BGM·SE
- EXP 800 / 도깨비 구슬 정산 +25 / 금화 60~90
- 드랍: 신물 "장산범의 송곳니"(20%, 회피 시 0.5s 그림자 대시 부여), 신화 조각 1
- BGM: P1 — `mb_jangsan_01.ogg`(현악 글리산도 + 산속 바람), P2 — `mb_jangsan_02.ogg`(타격감 강한 북 추가)
- SE 큐: 가짜 보물 출현 `se_fake_treasure.wav`, 측면 급습 `se_pounce.wav`, 위장 해제 포효 `se_tiger_roar_full.wav`

### MB02. 이무기 (Imugi / Proto-Dragon Serpent)

#### 2.1 메타
- **출전**: 한국 전설 — 천 년을 기다려 용이 되려는 큰 뱀. 여의주를 얻지 못하면 영원히 이무기로 남는다.
- **등장 챕터**: 2장 (신령의 숲)
- **등장 트리거**: 2장 스테이지 3:30 또는 봉인석 파괴.

#### 2.2 외형 & 컨셉
- 길이 6m 짙은 녹색 비늘 뱀. 머리에 미완성 뿔. 입에 청록색 미완성 여의주를 물고 있음.
- 컷인: 호수에서 물기둥 치솟으며 등장.

#### 2.3 기본 스탯
| 항목 | 값 |
|------|-----|
| HP | 2,800 |
| 방어력 | 4 |
| 이동속도 | 90 (몸통 슬라이드) |
| 기본 공격력(접촉) | 18 |
| 충돌박스 | 머리 56×56 / 몸통 마디 6개(각 48×48) |
| 스프라이트 | 머리 72×72, 마디별 64×64 |

> 머리만 피격 판정 ×1.0, 몸통 마디는 피격 판정 ×0.5, 꼬리는 ×0.25. 약점 명시.

#### 2.4 페이즈 구성
- **P1 (100%~50%)**: 몸통 활용 — 돌진/휩쓸기 콤보
- **P2 (50%~0%)**: 미완성 여의주 — 수속성 광역 + 물 장판

- SE 큐: 몸통 돌진 `se_serpent_charge.wav`, 잠수 `se_water_splash.wav`, 여의주 폭주 `se_orb_burst.wav`

### MB03. 차귀신 (Chagwishin / Cart-Riding Specter)

#### 3.1 메타
- **출전**: 제주/남부 민간 전승 — 죽은 자의 영혼이 수레를 끌고 산 자를 데려간다는 이야기.
- **등장 챕터**: 3장 (지하 황천)
- **등장 트리거**: 3장 스테이지 3:30 또는 봉인석 파괴.

#### 3.2 외형 & 컨셉
- 검은 망토 형체, 얼굴 없음. 낡은 나무 수레를 끌며 등장. 수레에서 차가운 흰 김.
- 컷인: 황천길 안개 속에서 수레바퀴 SE → 형체 드러남.

#### 3.3 기본 스탯
| 항목 | 값 |
|------|-----|
| HP | 3,800 |
| 방어력 | 6 |
| 이동속도 | 100 (수레 끌기) |
| 기본 공격력(접촉) | 22 |
| 충돌박스 | 보스 48×72 / 수레 96×56 |
| 스프라이트 | 보스 56×80, 수레 112×64 |

> 수레는 별도 파괴 가능 오브젝트(HP 1,200, 방어력 0). 파괴 시 P2 강제 진입.

#### 3.4 페이즈 구성
- **P1 (100%~50% 또는 수레 파괴 시)**: 수레 운용 — 직선 돌진 + 영혼 강제 탑승
- **P2 (50%~0%)**: 수레 없음 — 텔레포트 + 광역 곡소리

#### 3.5 페이즈별 패턴

**P1 패턴**

### 3.5 엔지니어 핸드오프 체크리스트

각 보스를 구현할 때 아래 항목을 모두 충족해야 한다.

- [ ] 페이즈 전환은 HP 비율 도달 시 즉시 발생(다음 이동/공격 직후가 아닌, 그 순간 강제 인터럽트). 단, 진행 중인 광역 데미지 판정은 1프레임 후 종료.
- [ ] 페이즈 전환 직후 보스 무적 + 다음 패턴 선택 쿨다운 0.5s 보장.
- [ ] 모든 텔레그래프는 시각 + 청각 SE 큐를 동시에 발생(접근성 — 색맹/난청 옵션 고려).
- [ ] 각 패턴은 발동 직전 1프레임 단위로 텔레그래프 충돌 영역이 화면에 실제로 그려져 있어야 한다(시각/판정 동기화).
- [ ] 보스 HP는 페이즈 전환 시점에서 정확히 해당 비율로 잠겨야 한다(누수 데미지 무시).
- [ ] BGM은 페이즈 전환 시 페이드아웃 0.4s + 다음 트랙 페이드인 0.4s로 크로스페이드.
- [ ] 회피·대시는 페이즈 전환 컷인 도중에도 입력 큐에 쌓여 컷인 종료 직후 발동 가능해야 한다.
- [ ] "최후의 ~" 패턴(흑룡 5%, 구미호 10% 등 1회성)은 보스 HP가 해당 임계 도달 시 다른 패턴을 중단하고 우선 발동.
- [ ] 모든 보스 처치 시 신물 드랍률은 메타 강화 "행운" 레벨 영향을 받지 않는다(보스 드랍은 별도 풀).
- [ ] 컷인은 스킵 옵션 제공(2회차 이상 또는 옵션 ON 시).
