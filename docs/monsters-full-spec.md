## 0. 사용 규약

### 0.1 명세 형식

각 몬스터는 다음 8개 항목을 모두 포함한다.

1. **이름**: 한글 + 영문 로마자 표기 (in-game ID로 사용)
2. **출전**: 신화/설화 출처 (가능한 한 문헌명 명시 — 삼국유사, 조선왕조실록, 어우야담, 동국세시기, 민간 설화 등)
3. **외형 설명**: 픽셀아트 제작 시 핵심 실루엣과 색감
4. **행동 패턴**: 실제 AI 상태머신으로 변환 가능한 수준의 의사 코드 수준 설명
5. **스탯 블록**: HP / 이동속도 / 접촉 데미지 / 원거리 데미지 / 공격 쿨다운 / 사거리 / 처치 EXP / 금화 드롭 확률
6. **특수 능력**: 발동 조건 + 효과 + 쿨다운 (없으면 "없음")
7. **출현 챕터**: 1~5 또는 히든
8. **스프라이트 크기**: 픽셀 단위 (가로 × 세로)

### 0.2 스탯 기준선 (Baseline)

플레이어 표준값 (캐릭터 미강화, 메타 강화 0레벨 기준)을 기준으로 설계한다.

| 항목 | 값 |
|------|-----|
| 플레이어 HP | 100 |
| 플레이어 기본 공격력 | 10 (1초당 1회) |
| 플레이어 이동속도 | 120 px/s |
| 1프레임 | 1/60 s |

챕터별 일반 몬스터 평균 스탯 스케일링:

| 챕터 | HP 배율 | 데미지 배율 | 이동속도 배율 | 비고 |
|------|---------|-------------|---------------|------|
| 1 | 1.0 | 1.0 | 1.0 | 입문 |
| 2 | 1.6 | 1.3 | 1.1 | 숲 |
| 3 | 2.4 | 1.7 | 1.0 | 황천 |
| 4 | 3.8 | 2.3 | 1.2 | 천상 |
| 5 | 6.0 | 3.0 | 1.3 | 최종 |
| 히든 | 변동 | 변동 | 변동 | 카오스 — 개별 명세 |

같은 챕터 안에서도 스테이지 진행 시간(0~5분)에 따라 동일 몬스터의 스탯이 선형 증가한다(체력 ×1.0→×1.5, 데미지 ×1.0→×1.3).

### 0.3 단위

- HP, 데미지: 정수
- 이동속도: 픽셀/초 (px/s)
- 쿨다운, 사거리: 각각 초(s), 픽셀(px)
- 스프라이트 크기: 픽셀 (가로×세로), 충돌박스는 별도 명시 없으면 스프라이트 크기 ×0.8

### 0.4 AI 상태(공통)

모든 몬스터는 아래 공통 상태를 가지며, 행동 패턴 항목에서 추가/변형을 명시한다.

```
Spawn → Idle → Pursue → Attack → (Cooldown) → Pursue
                  ↘ Stunned → Pursue
                  ↘ Dying → Despawn
```

기본 시야 반경(Detection Radius)은 별도 명시 없으면 800 px이다.

## 1. 챕터 1 — 두멍마을 (9종)

### M01. 도깨비불 (Dokkaebi-bul / Goblin Fire)

- **출전**: 민간 전승, 한국 야담 — 밤길 나그네를 홀리는 푸른 불꽃
- **외형**: 푸르스름한 도깨비불 구체, 작은 눈 한 쌍, 꼬리처럼 흩날리는 잔불. 색상 #5BCEFA 베이스, 외곽 노란 발광.
- **행동 패턴**:
  - Idle: 공중에서 위아래로 ±8px 사인파 부유
  - Pursue: 플레이어 위치로 직선 추적 (꺾임 없음)
  - Attack: 접촉 시 데미지 발생, 즉시 0.5초 후퇴 후 재추적
- **스탯**: HP 10 / 이속 50 / 접촉 3 / 원거리 0 / 쿨다운 1.0s / 사거리 - / EXP 2 / 금화 8%
- **특수 능력**: 없음
- **출현 챕터**: 1~5 (가장 흔한 유닛, 챕터 스케일링 적용)
- **스프라이트**: 16×16

### M02. 달걀귀신 (Dalgyal-gwisin / Egg Ghost)

- **출전**: 학교 괴담, 전래 요괴 — 얼굴 없는 달걀 모양 귀신
- **외형**: 달걀형 흰색 몸체, 얼굴 자리가 매끄러운 표면. 옅은 회색 그림자, 작은 발은 없음(부유).
- **행동 패턴**:
  - 항상 3~5마리 무리(Group)로 스폰
  - 무리의 중심점이 플레이어 추적 (각 개체는 중심점에서 ±20px 오프셋)
  - 플레이어가 50px 이내 진입 시 무리 전체가 일제히 돌진 (이속 ×1.6, 0.8초 지속)
- **스탯**: HP 8 / 이속 70 / 접촉 4 / 원거리 0 / 쿨다운 0.8s / 사거리 - / EXP 2 / 금화 6%
- **특수 능력**: **돌진**(Charge) — 플레이어 50px 진입 시 무리 일제 돌진, 쿨다운 4초.
- **출현 챕터**: 1, 2장
- **스프라이트**: 20×24

### M03. 물귀신 (Mul-gwisin / Water Ghost)

- **출전**: 한국 물귀신 전설 — 익사자의 영혼이 산 자를 물속으로 끌어들임
- **외형**: 젖은 머리카락이 얼굴을 덮은 청록색 형체, 하반신이 물웅덩이와 연결된 듯 흐릿.
- **행동 패턴**:
  - 맵의 물 웅덩이 환경 오브젝트에서만 스폰 (없으면 5초마다 임시 웅덩이 생성)
  - 웅덩이 반경 150px 내에서만 활동 (벗어나면 자체 데미지)
  - 플레이어가 200px 이내 진입 시 손을 뻗어 끌어당김(이속 디버프 부여)
- **스탯**: HP 18 / 이속 35 / 접촉 5 / 원거리 0 / 쿨다운 2.0s / 사거리 - / EXP 4 / 금화 15%
- **특수 능력**: **끌어당기기**(Drag) — 200px 이내 플레이어에게 이속 -40% 디버프 3초 부여, 쿨다운 5초.
- **출현 챕터**: 1, 3장
- **스프라이트**: 32×48

### M04. 어둑시니 (Eodukshini / Darkness Spirit)

- **출전**: 한국 민간 설화 — 보면 볼수록 커지는 어둠 속의 형체
- **외형**: 검은 그림자 인영, 빨간 점눈 두 개. 초기에는 24×24, 플레이어 시선 안에 있으면 점점 커짐.
- **행동 패턴**:
  - 시야 안(플레이어 정면 120° 부채꼴)에 들어가면 매 초마다 스프라이트 ×1.1 확대 (최대 ×2.0)
  - 시야 밖일 때는 빠르게 접근(이속 ×1.5)
  - 크기에 비례하여 데미지 증가 (×1.0~×2.0)
- **스탯**: HP 14 / 이속 45 / 접촉 4 / 원거리 0 / 쿨다운 1.2s / 사거리 - / EXP 3 / 금화 10%
- **특수 능력**: **응시 증대**(Gaze Growth) — 플레이어 시야 내에 있는 동안 매 초 크기/데미지 ×1.1, 최대 ×2.0. 시야에서 벗어나도 크기 유지(처치 또는 스테이지 종료까지).
- **출현 챕터**: 1, 2장
- **스프라이트**: 24×24 (가변, 최대 48×48)

### M05. 그슨대 (Geuseundae / Stretching Spirit)

- **출전**: 제주 민간 설화 — 길에서 보면 자꾸 키가 길어지는 귀신
- **외형**: 길쭉한 회백색 몸체, 머리가 위로 늘어남. 정수리에 떨어진 갓을 얹고 있음.
- **행동 패턴**:
  - 멀리서는 짧은 형태로 천천히 추적
  - 플레이어와의 거리가 250px 미만이 되면 매 0.5초마다 세로 스프라이트 +6px (최대 64px)
  - 길어질수록 머리 위에서 내려치는 원거리 공격 가능 (사거리 = 현재 길이 × 2)
- **스탯**: HP 16 / 이속 55 / 접촉 4 / 원거리 5 / 쿨다운 2.0s / 사거리 가변 / EXP 3 / 금화 10%
- **특수 능력**: **신장 연장**(Stretch) — 거리 250px 미만 시 길이 증가. 길이가 48px 이상일 때 머리 내려치기(원거리 공격) 활성화, 쿨다운 2.0s.
- **출현 챕터**: 1, 2장
- **스프라이트**: 16×24 (가변, 최대 16×64)

### M06. 빗자루귀신 (Bitjaru-gwisin / Broom Ghost)

- **출전**: 한국 가신(家神) 신앙 — 오래된 빗자루가 깃들어 변한 요괴
- **외형**: 거꾸로 선 빗자루, 빗자루 솔 끝에 작은 얼굴. 짚 색 본체에 빨간 띠.
- **행동 패턴**:
  - 회전하며 추적 (시각적으로 1초당 1바퀴 회전)
  - 5초마다 멈춰서 솔을 펼쳐 주변 64px에 짧은 광역 공격
  - 광역 공격 직전 0.5초 동안 정지(예고 모션)
- **스탯**: HP 12 / 이속 60 / 접촉 3 / 원거리 4 / 쿨다운 5.0s / 사거리 64 / EXP 3 / 금화 12%
- **특수 능력**: **먼지 회오리**(Dust Whirl) — 5초마다 솔 펼쳐 자신 주변 64px 도넛형 광역 데미지(내경 24, 외경 64), 0.5초 예고. 명중 시 0.5초 시야 흐림 디버프.
- **출현 챕터**: 1장
- **스프라이트**: 20×32

### M07. 손각시 (Songakshi / Maiden Ghost)

- **출전**: 한국 원귀 설화 — 결혼 못하고 죽은 처녀의 작은 한(恨)
- **외형**: 흰 소복을 입은 작은 처녀, 머리카락이 얼굴을 가림. 양손에 빨간 댕기.
- **행동 패턴**:
  - 평소엔 천천히 떠도는 비공격 상태
  - 플레이어와 거리 300px 이내가 되면 5초 동안 추적, 닿으면 강한 단발 공격 후 1.5초간 도주
  - HP 50% 이하가 되면 0.3초 깜빡임 후 순간이동(±100px 랜덤)
- **스탯**: HP 20 / 이속 50 / 접촉 6 / 원거리 0 / 쿨다운 2.5s / 사거리 - / EXP 4 / 금화 14%
- **특수 능력**: **점멸**(Blink) — HP 50% 이하 진입 시 ±100px 랜덤 워프, 쿨다운 4.0s.
- **출현 챕터**: 1, 3장
- **스프라이트**: 24×40

### M08. 몽달귀신 (Mongdal-gwisin / Bachelor Ghost)

- **출전**: 한국 민간 설화 — 장가 못 가고 죽은 총각의 한
- **외형**: 다 헤진 도포 차림의 총각 영혼, 갓이 비뚤어짐. 머리 위에 종이꽃 한 송이.
- **행동 패턴**:
  - 손각시와 같은 화면에 동시 스폰되면 두 마리 모두 이속 ×1.3 (서로 다가가려는 듯)
  - 단독 시: 일반 추적
  - 공격 시 종이꽃을 던지는 원거리 투사체 (직선)
- **스탯**: HP 18 / 이속 55 / 접촉 4 / 원거리 5 / 쿨다운 1.8s / 사거리 200 / EXP 4 / 금화 12%
- **특수 능력**: **외로움**(Loneliness) — 손각시와 같은 화면 존재 시 양쪽 이속 +30%, 양쪽 공격력 +20%.
- **출현 챕터**: 1, 3장
- **스프라이트**: 24×40

### M09. 두두리 (Duduri / Drum Spirit)

- **출전**: 삼국유사 권1 「비형랑조」 — 도깨비 무리의 우두머리로 등장하는 신이한 존재
- **외형**: 빨간 머리띠를 한 작은 도깨비, 양손에 작은 북(소고). 색감은 황토와 빨강.
- **행동 패턴**:
  - 플레이어 주변 200~300px 거리를 유지(원거리)
  - 매 2초마다 북을 치면 주변 100px 반경에 음파 충격(데미지 + 0.3초 넉백)
  - 다른 1장 몬스터 5마리 이내에 있을 시 그 몬스터들의 이속 +25% 버프
- **스탯**: HP 22 / 이속 45 / 접촉 3 / 원거리 5 / 쿨다운 2.0s / 사거리 100 / EXP 5 / 금화 18%
- **특수 능력**: **북소리 진군**(Drum March) — 자신 반경 240px 안의 챕터1 일반 몬스터에게 이속 +25% 오라(지속).
- **출현 챕터**: 1장
- **스프라이트**: 28×32

## 2. 챕터 2 — 깊은 숲 (10종)

### M10. 삼두구 (Samdugu / Three-headed Hound)

- **출전**: 한국 삼두견(三頭犬) 전설 — 세 개의 머리를 가진 신령한 사냥개의 타락형
- **외형**: 검은 털의 거대 개, 머리 셋 (가운데는 정면, 좌우 양옆). 각 눈은 붉음.
- **행동 패턴**:
  - 직선 돌진형(이속 ×1.3 부스트 1초)
  - 사거리 안에서 3방향(정면/좌45°/우45°) 동시 짖기(음파) 공격
  - 짖을 때 0.4초 정지
- **스탯**: HP 35 / 이속 75 / 접촉 8 / 원거리 9 / 쿨다운 3.0s / 사거리 220 / EXP 8 / 금화 22%
- **특수 능력**: **삼중 포효**(Triple Howl) — 3방향 동시 부채꼴(각 30°) 음파, 쿨다운 3.0s. 명중 시 0.5초 스턴.
- **출현 챕터**: 2, 3장
- **스프라이트**: 48×40

### M11. 호랑이 망령 (Horangi-mangryeong / Tiger Wraith)

- **출전**: 한국 산신 신앙 — 산을 다스리던 호랑이가 한을 품고 망령이 됨
- **외형**: 반투명한 푸른빛 호랑이, 줄무늬는 흰빛. 발자국에 푸른 잔영.
- **행동 패턴**:
  - 매복형: 시야 밖에서 출현 후 일직선 돌격(이속 ×2.5, 1.5초 지속)
  - 돌격 후 1.5초간 정지(스턴 가능)
  - 정지 중 피격받으면 짧게 포효(주변 80px 적/플레이어 0.5초 둔화)
- **스탯**: HP 40 / 이속 70 (돌격 시 175) / 접촉 12 / 원거리 0 / 쿨다운 4.0s / 사거리 - / EXP 10 / 금화 30%
- **특수 능력**: **매복 돌격**(Ambush Charge) — 시야 밖 200px 거리에서 출현 후 즉시 직선 돌격. 충돌 또는 1.5초 후 정지. 쿨다운 6초.
- **출현 챕터**: 2장
- **스프라이트**: 48×32

### M12. 멧돼지 요괴 (Metdwaeji-yogoe / Boar Yokai)

- **출전**: 한국 민간 설화 — 산속 늙은 멧돼지가 요력을 얻어 사람을 해침
