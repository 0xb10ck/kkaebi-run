**기본 무기 상세** — 도깨비 방망이 (회전 타격)
- 캐릭터 주위 반경 **70px** 원형 영역을 회전 타격
- 1회 발동 시 데미지 = ATK × **1.0**
- 다중 타격: 영역 내 적 최대 6마리 동시 타격 (관통 아님, 영역기)
- 발동: 자동, 1.0초 주기
- 시각: 회색 방망이가 캐릭터 중심으로 한 바퀴 회전 (잔상 청색)

### 1.3 고유 패시브 — 도깨비 상인의 손맛

| 항목 | 내용 |
|------|------|

| 적용 대상 | 모든 몬스터(미니보스·보스·엘리트 포함). 환경 보상(상자 등)에는 미적용. |
| 효과 발동 시점 | 몬스터 사망 판정 직후, 드랍 테이블 계산 단계 |

> **밸런스 메모**: 도깨비 시장(GDD §8.1)·런 종료 정산(GDD §7.3) 양쪽에서 가치를 발휘하는 *이코노미 캐릭터*. 절대 DPS는 낮지만 도깨비 구슬 누적 속도가 빠름 → 메타 진행 가속.

### 1.5 시작 스킬 풀 (Starting Skill Pool / Weights)

런 시작 시 **레벨업 3택 첫 등장 가중치**가 다음과 같이 보정된다(평균치 대비). 기본 가중치 1.0을 기준으로:

| 스킬 ID (skills-full-spec.md 기준) | 속성/등급 | 가중치 | 비고 |
|--------------------------------------|-----------|--------|------|
| 흙벽 (TO-01) | 토/일반 | **×2.0** | 균형형 시작 보조 |
| 금속 사슬 (GE-01) | 금/일반 | **×2.0** | 무기 시너지 |
| 신목의 가호 (MO-03) | 목/고급 | ×1.3 | 회복 보조 |
| 자석장(가칭 PU 증가 패시브) | 무속성 패시브 | **×1.5** | PU 강화 친화 |
| 그 외 30종 | — | ×1.0 | 정상 풀 |

> **시작 보유 스킬**: 없음(기본 무기만 보유). 캐릭터 6종 중 *유일하게 시작 시 액티브 스킬을 따로 들고 시작하지 않는 균형형*. R1 첫 레벨업에서 위 가중치에 따라 3택 제시.

### 1.6 친밀도 스킬트리 — **분기형 (T자 + 2갈래)**

> **구조**: 공통 루트 노드 → 좌(자원형) / 우(전투형) **2갈래** → 각 갈래 끝에 **각성 노드** 1개. 총 **10 노드** (루트 2 + 좌4 + 우4).
> **해금 통화**: 친밀도 레벨업(런 종료 시 사용 횟수에 따라 1~3 증가) + 도깨비 구슬.

| 노드 | 명칭 | 효과 | 잠금 해제 조건 |
|------|------|------|----------------|
| **T1** (루트) | 노련한 손목 | 도깨비 방망이 회전 반경 +10% (70 → 77px) | 친밀도 Lv.1 |
| **T2** (루트) | 호랑이 가죽의 인내 | HP +10 | 친밀도 Lv.2, 도깨비 구슬 10 |
| **L1** (자원 분기) | 금화 자석 | PU 반경 +15px (금화 전용) | 친밀도 Lv.3, T2 해금 |
| **L2** | 상인의 직감 | 금화 획득량 추가 +10% (패시브 합산 시 +30%) | 친밀도 Lv.4, L1 해금, 누적 처치 3,000 |
| **L3** | 도깨비 시장 단골 | 도깨비 시장(GDD §8.1) 등장 확률 +30% | 친밀도 Lv.5, L2 해금, 도깨비 구슬 50 |
| **L4 (각성)** | 방망이의 황금 일격 | 회전 타격 처치 시 **5% 확률로 즉시 금화 5개 드랍**(이 드랍은 패시브 +20% 적용 안 함) | 친밀도 MAX, L3 해금, 챕터 3 보스 단독 클리어(다른 캐릭터로 불가) |
| **R1** (전투 분기) | 강철 손잡이 | ATK +1 (10 → 11) | 친밀도 Lv.3, T1 해금 |
| **R2** | 빠른 손놀림 | AS -0.10s (1.00 → 0.90s) | 친밀도 Lv.4, R1 해금, 누적 처치 2,000 |
| **R3** | 방망이 폭풍 단축 | 궁극기 쿨다운 -10s (45 → 35s) | 친밀도 Lv.5, R2 해금, 챕터 2 보스 클리어 |
| **R4 (각성)** | 폭풍의 잔향 | 궁극기 종료 후 5초간 일반 공격 회전 반경 두 배 | 친밀도 MAX, R3 해금, 한 런에서 궁극기 5회 발동 |

> **상호 배타 규칙**: L4(자원 각성)와 R4(전투 각성) **둘 다 해금 가능**하지만, 한 런에서는 시작 시 **하나만 선택**(런 시작 메뉴에서 토글).

### 1.7 밸런스 코멘트

- **역할**: *이코노미 균형형 (Economy / Balanced)*. 절대 DPS는 6종 중 최하위권이지만 메타 진행 속도가 가장 빠르다.
- **상성 좋은 스킬 트리**: `skills-full-spec.md §9.3`에 따라 **토/금 중심**. 흙벽(TO-01) → 철벽 성채(진화)로 단단한 회전 타격 빌드 추천.
- **상성 좋은 챕터**: 챕터 1~3. 단단한 보스 페이즈에서 회복·둔화 부족으로 챕터 4 이후 후반부에 적응 어려움 → R4 각성 또는 신목의 가호 의존도 ↑.
