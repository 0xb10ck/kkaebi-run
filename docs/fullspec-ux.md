### 0.3 공통 컴포넌트

- **TitleBar**: 좌측 24×24 뒤로가기 화살표(없는 경우 홈 아이콘) + 중앙 붓글씨 제목 + 우측 정보(현재 보유 구슬/이파리).
- **ResourceChip**: 16×16 아이콘 + 숫자, 우상단 고정. 도깨비 구슬(`#FFD700`) · 신목 이파리(`#4CAF50`) · 신화 조각(`#9C27B0`).
- **HanjiCard**: 한지 텍스처 + 1px `fg/ink` 외곽 + 4모서리 매듭 장식 4×4px.
- **PrimaryButton**: 나무 프레임(`bg/wood`) + 흰 글씨, 비활성 시 회색 마스크 + 우상단 자물쇠 아이콘.
- **ProgressBar**: 24px 높이, 좌측 텍스트, 우측 비율. 채워진 부분은 용도별 색(영구 강화=황금, HP=적, 경험=청).

### 1.3 신목 시각화 (메인 메뉴 상단)

```
Lv.0 시들음 :  ▓▓░░     검은 둥치 + 그을음(망점)
Lv.1 새싹   :  ▓▓Y░     연두 새싹 1개
Lv.2 묘목   :  ▓⌒⌒▓     얇은 줄기 + 잎 5장
Lv.3 어린나무: ╲▓╱      가지 3분기, 잎 무성
Lv.4 큰나무 : ╲⌘╱      꽃봉오리 + 5m 윤곽
Lv.5 거목   : ╲✪╱      황금 잎 + 후광
Lv.6 신목   : ╲★╱+오로라  맥박 발광 4프레임 루프
```
- **위치**: 화면 중앙 X, Y=96, 220×220px.
- **진행도 게이지 위치**: 신목 아래 Y=332, 360×24px. 좌측 "Lv.{현재} → Lv.{다음}", 우측 "{현재구슬} / {필요구슬}".
- **이파리 표시**: 게이지 바로 아래 Y=364. 필요 개수만큼 ●(채움 `#4CAF50`)/○(빔 `bg/paper-dark`). 부족 시 ○ 옆 "🍃 부족"(`#FF3B30`).
- **상호작용**: 신목 그래픽 자체를 탭하면 §4 신목 화면으로 이동(메뉴 카드 4와 동일 액션).
- **호버 툴팁** (PC): 다음 단계 해금 컨텐츠 3줄 미리보기.

### 6.4 보상 수령 인터랙션

1. [수령하기] 탭 → 0.2초 압축 → 0.4초 동안 보상 아이콘(구슬·이파리·신화 조각)이 카드에서 우상단 ResourceChip으로 날아감 (포물선) → 카드 상태 "수령 완료"로 전환 + 토스트 1.5초 "보상을 수령했습니다."
2. 칭호·스킨 보상은 별도 모달 "새로운 칭호를 획득했습니다: {이름}" + [확인].
3. 일/주간 갱신 직후 미수령 → 자동 우편함으로 이동 (5개 보관 한도).

- **포터블 해상도**: 본 명세는 480×854 portrait 기준. 데스크탑(960×540 landscape — art-style-guide §6.5)은 메인 메뉴를 가로형으로 재배치(2열 → 4열 그리드, 신목 좌측). 동일 데이터, 다른 레이아웃.

### 12.2 QA 인계 — UX 검증 기준

- [ ] 7개 진입점 카드가 480×854에서 안전 영역 내에 표시되고 모든 텍스트가 잘리지 않음 (한글 인코딩 UTF-8 검증)
- [ ] 신목 6단계 외형이 단계 업 시 정확히 모핑하고 진행도 게이지가 백트래킹 없이 단방향
- [ ] 캐릭터 6종 카드가 잠금/해금/선택 3상태로 명확히 구분
- [ ] 영구 강화 8항목 카드가 GDD §7.3 수치와 정확히 일치 (만렙 + 누적 효과)
- [ ] 도감 미해금 카드는 항상 실루엣 + `?`, 챕터 힌트만 노출 (처치 조건 비공개)
- [ ] 도전과제 카드 4상태(잠금/진행/완료/수령)가 명확히 구분
- [ ] 챕터/스테이지 노드 트리가 1-1 → 1-2 → 1-3 → 보스 순으로 잠김 해제
- [ ] 보스 컷인 0.5s+1.0s+0.3s 타이밍 정확히 측정
- [ ] 보스 HP 바 페이즈별 색 변화(적→주황→위험 적) 적용
- [ ] 막간에서 [다음 챕터로] 자동 1:30 카운트다운 동작
- [ ] 모든 사용자 노출 문구가 한국어 존댓말 (§11 카피 표 위반 0건)
- [ ] 텔레그래프는 art-style-guide §1.3.4 7색 + 형태 + SE 3중 신호 (색맹 옵션 호환)
- [ ] 일시정지 → [메인 메뉴로] 시 확인 모달 표시, 사망 정산 규칙(구슬 30%) 적용
- [ ] 도전과제 수령 아이콘이 ResourceChip으로 포물선 비행
- [ ] 모든 화면에서 [←] 뒤로가기가 동작하고 데이터 손실 없음

### 12.3 우선순위 (구현 순서 권장)

1. **Phase 1 (MVP — 인게임 1회 완주 가능)**: 메인 메뉴(7카드 + 신목 외형 Lv.0~1) → 캐릭터 선택(뚝딱이 단일) → 챕터/스테이지 선택(1장만) → 보스 컷인 1종 + HP 바 → 막간(상점 2종, 강화 단순)
2. **Phase 2 (챕터 1 완성 + 메타 진입)**: 신목 Lv.0~3 + 영구 강화 8항목 + 도감 등록 시스템 + 도전과제 일회성 5종
3. **Phase 3 (콘텐츠 확장)**: 캐릭터 6종 + 친밀도 트리 모달 + 도감 100% + 도전과제 35종 + 챕터 2~5 + 히든 + 페이즈 전환 컷인 + 일일/주간 도전 갱신 로직
4. **Phase 4 (폴리싱)**: 신목 단계 업 컷인 3s + 보스 처치 슬로모션 + 정산 화면 + 설정 화면 + 색맹/접근성 옵션 + 모바일 진동

### 12.4 확장 메모

- meta-systems-spec.md의 26노드 트리는 **고급 트리** 진입 화면으로 분리(본 명세는 GDD §7.3의 8항목만). 26노드 트리 UX는 Phase 3에서 별도 명세.
- 일시정지 메뉴, 정산 화면, 설정 화면, 패배 화면, 우편함 화면, 진엔딩 컷신, 계절 이벤트 화면, 무한 모드 화면은 본 문서 범위 외 — 향후 명세.

## UX Spec

### User flow
1. 사용자가 앱을 켜면 스플래시 → 메인 메뉴로 이동합니다.
2. 메인 메뉴 상단의 신목 시각화로 현재 메타 진행도(Lv.{n}/이파리/구슬)를 한눈에 확인합니다.
3. 7개 진입점 중 하나를 탭해서 캐릭터 선택, 영구 강화, 신목, 도감, 도전과제, 설정으로 이동하거나 [시작하기]로 챕터 선택 → 스테이지 선택 → 인게임으로 진행합니다.
4. 스테이지를 모두 클리어하면 보스 컷인(0.5+1.0+0.3s) → 보스전 → 처치 컷인 → 막간(상점/강화) → 다음 챕터 또는 런 종료를 진행합니다.

### Screen list
- 메인 메뉴 (신목 + 7카드)
- 캐릭터 선택 (6캐릭터 카드 + 상세 + 친밀도 트리 모달)
- 영구 강화 (8항목 그리드)
- 신목 (6단계 시각화 + 게이지 + 단계 업 미리보기)
- 도감 (요괴/신물/장소 3탭 + 상세 모달)
- 도전과제 (일회성/일일/주간 3탭)
- 챕터 선택 (1~5장 + 히든)
- 스테이지 선택 (1-1~1-3 + 보스 노드 트리)
- 보스 등장 컷인 (한자 + 글로우)
- 보스 HP 바 / 페이즈 전환 컷인 / 보스 처치 컷인
- 막간 (상점/강화 2탭)
- 일시정지 모달 / 설정 (보조)

### Key states
- **loading**: 한지 + 붓글씨 "잠시만 기다려 주세요…"
- **empty**: 신규 유저 — 신목 Lv.0 + "첫 챕터를 도전해 주세요" 가이드
- **blocked**: 회색 마스크 + 자물쇠 + "신목 Lv.{n}을 달성하면 열립니다."
- **awaiting approval**: 카드 황금 외곽 발광 + 메인 메뉴 진입점 빨간 점 (수령 가능 도전과제)
- **completed**: 만렙·클리어 카드 황금 외곽 + ★ 또는 ✓ 표시
- **error**: "데이터를 불러오지 못했습니다. 다시 시도해 주세요." + [재시도]

### Interaction rules
- 모든 UI 호버/포커스 시 1px 황금 외곽(`#FFD700`) 발광 (art-style-guide §6.8).
- 모든 클릭/탭은 0.1초 스케일 0.95 압축 + SE 1종 (UI 카테고리).
- 컷인 중에는 입력 차단 (보스 등장 1.8s, 페이즈 전환 1.2s, 처치 2.2s).
- 텔레그래프는 색 + 형태 + SE 3중 신호 (색맹 옵션 호환).
- 신목 진행도는 단방향(백트래킹 불가) — 단계는 절대 내려가지 않습니다.
- 영구 강화 [강화하기]는 자원 충족 시에만 활성, 만렙 시 "✓ 만렙" 라벨로 치환.
- 도전과제 보상 수령은 즉시 ResourceChip으로 포물선 비행 + 토스트 1.5s.
- 화면 전환은 슬라이드 좌(진입), 슬라이드 우(뒤로), 페이드(컷인) 3종으로 통일.
- 모든 모달은 ESC 또는 화면 외 탭으로 닫힘, 단 컷인/페이즈 전환은 닫힘 불가.
